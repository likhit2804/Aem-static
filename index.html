Lecture 1: Introduction to Differential Equations

What is a Differential Equation?

An equation that expresses a relationship between functions, their derivatives, and variables is known as a differential equation. For example, a function \(y\) depends on a variable \(x\).

Classification of Differential Equations

Differential equations (DEs) can be classified into two main types:

  • Ordinary Differential Equations (ODEs): An ODE is a DE where the dependent variable (e.g., \(y\)) is dependent on only one independent variable (e.g., \(x\)). The general form is \(F(x,y,y',y'',y'''...) = 0\).
    • Example: \(\frac{dy}{dx} = \sin x + \cos x\).
  • Partial Differential Equations (PDEs): A PDE is a DE where the dependent variable (e.g., \(y\)) is dependent on two or more independent variables (e.g., \(x, t\)). The general form is \(F(x,t,y,\frac{\partial y}{\partial x},\frac{\partial y}{\partial t}, ...) = 0\).
    • Example: \(\frac{\partial y}{\partial x} + t\frac{\partial y}{\partial t} = 2y\).

Order and Degree of a Differential Equation

  • Order: The order of a DE is defined as the order of the highest derivative present in the equation.
  • Degree: The degree of a DE is the power of the highest-order derivative after the equation has been cleared of any radicals and fractions with respect to the derivatives.
Examples: Find the Order and Degree

1. \(\frac{d^{2}y}{dx^{2}} = 0\)

Answer: Order = 2, Degree = 1.

2. \(\frac{d^{2}y}{dx^{2}} + m^2y = 0\)

Answer: Order = 2, Degree = 1.

3. \(y=x\frac{dy}{dx}+\frac{a}{\frac{dy}{dx}}\)

Steps: First, clear the fraction by multiplying by \(\frac{dy}{dx}\). This gives \(y\frac{dy}{dx} = x(\frac{dy}{dx})^2 + a\). The highest derivative is \(\frac{dy}{dx}\) (order 1), and its highest power is 2.
Answer: Order = 1, Degree = 2.

4. \(y\frac{d^{2}y}{dx^{2}} - (\frac{dy}{dx})^2 = y^2 \log y\)

Answer: Order = 2, Degree = 1.

5. \([1+(\frac{dy}{dx})^{2}]^{3/2}=a\frac{d^{2}y}{dx^{2}}\)

Steps: Square both sides to remove the radical: \([1+(\frac{dy}{dx})^{2}]^{3}=a^2(\frac{d^{2}y}{dx^{2}})^2\). The highest derivative is \(\frac{d^{2}y}{dx^{2}}\) (order 2), and its power is 2.
Answer: Order = 2, Degree = 2.

Formation and Solution of DEs

DEs are formed to eliminate arbitrary constants from an equation. A key principle is that the order of the resulting DE is equal to the number of constants to be eliminated.

Example: Form a DE from \(y = ax^2 + bx\)

The equation has two constants, \(a\) and \(b\). So we need a 2nd order DE.

1. Differentiate once: \(\frac{dy}{dx} = 2ax + b\).

2. Differentiate again: \(\frac{d^2y}{dx^2} = 2a\).

3. From step 2, we get \(a = \frac{1}{2} \frac{d^2y}{dx^2}\).

4. Substitute \(a\) back into the first derivative to find \(b\): \(b = \frac{dy}{dx} - 2ax = \frac{dy}{dx} - x\frac{d^2y}{dx^2}\).

5. Substitute \(a\) and \(b\) back into the original equation: \(y = (\frac{1}{2}\frac{d^2y}{dx^2})x^2 + (\frac{dy}{dx} - x\frac{d^2y}{dx^2})x\).

6. Simplifying gives the final DE: \(y = x\frac{dy}{dx} - \frac{1}{2}x^2\frac{d^2y}{dx^2}\), which has order 2 and degree 1.

Types of Solutions

A solution is a relation between variables that satisfies the DE.

  • General Solution: A solution where the number of arbitrary constants equals the order of the DE. Ex: For \(\frac{dy}{dx}-2y=0\), the general solution is \(y=ce^{2x}\).
  • Particular Solution: Obtained from the general solution by assigning specific values to the constants. Ex: \(y=2e^{2x}\) is a particular solution.
  • Singular Solution: A solution that is neither general nor particular. Only some equations have them. Ex: For \((\frac{dy}{dx})^2 - 4y = 0\), \(y=0\) is a singular solution.

First Order, First Degree DEs

These equations are of the form \(F(x, y(x), \frac{dy}{dx}) = 0\). They are classified into several types, including:

  • Variable Separable Equations
  • Homogeneous Equations
  • Linear DEs
  • Exact Equations
  • And equations reducible to these forms.

1. Variable Separable Equations

If a DE can be rearranged so that all functions of \(x\) and \(dx\) are on one side, and all functions of \(y\) and \(dy\) are on the other, the variables are separable. The general form is \(f(x)dx = g(y)dy\). The solution is found by integrating both sides: \(\int f(x)dx = \int g(y)dy + c\).

Example: Solve \((1-x)dy + (1-y)dx = 0\)

1. Rearrange the equation: \((1-x)dy = -(1-y)dx = (y-1)dx\).

2. Separate the variables: \(\frac{1}{y-1}dy = \frac{1}{1-x}dx\).

3. Integrate both sides: \(\int \frac{1}{y-1}dy = \int \frac{1}{1-x}dx\).

4. Result of integration: \(\log(y-1) = -\log(1-x) - c\).

5. Combine log terms: \(\log(y-1) + \log(1-x) = -c\), which gives \(\log((y-1)(1-x)) = -c\).

6. Exponentiate to get the general solution: \((y-1)(1-x) = e^{-c}\). Let \(A = e^{-c}\), so \((y-1)(1-x) = A\).

Lecture 2: Methods for Solving First Order DEs

2. Equations Reducible to Variable Separable

Some first-order, first-degree DEs can be simplified into a variable separable form by using a suitable substitution.

Example: Solve \(\frac{dy}{dx} = (4x+y+1)^2\) given \(y(0)=1\)

1. Substitution: Let \(t = 4x+y+1\).

2. Differentiate the substitution w.r.t x: \(\frac{dt}{dx} = 4 + \frac{dy}{dx}\), so \(\frac{dy}{dx} = \frac{dt}{dx} - 4\).

3. Substitute back into the DE: \(\frac{dt}{dx} - 4 = t^2\), which simplifies to \(\frac{dt}{dx} = t^2 + 4\).

4. Separate variables: \(\frac{dt}{t^2+4} = dx\). This is now a variable separable equation.

5. Integrate both sides: \(\int \frac{dt}{t^2+2^2} = \int dx\).

6. General Solution: This gives \(\frac{1}{2}\tan^{-1}(\frac{t}{2}) = x+c\). Substituting back for \(t\), we get \(\frac{1}{2}\tan^{-1}(\frac{4x+y+1}{2}) = x+c\).

7. Find Particular Solution: Use the condition \(y(0)=1\), so when \(x=0\), \(y=1\).
\(\frac{1}{2}\tan^{-1}(\frac{4(0)+1+1}{2}) = 0+c \implies \frac{1}{2}\tan^{-1}(1) = c\).
This gives \(c = \frac{1}{2}(\frac{\pi}{4}) = \frac{\pi}{8}\).

8. Final Answer: The particular solution is \(\frac{1}{2}\tan^{-1}(\frac{4x+y+1}{2}) = x + \frac{\pi}{8}\).

3. Homogeneous Differential Equations

A DE of the form \(\frac{dy}{dx} = \frac{f(x,y)}{g(x,y)}\) is homogeneous if \(f(x,y)\) and \(g(x,y)\) are homogeneous functions of the same degree. A function \(F(x,y)\) is homogeneous of degree \(n\) if \(F(tx, ty) = t^n F(x,y)\).

To solve, use the substitution \(y=vx\). This leads to \(\frac{dy}{dx} = v + x\frac{dv}{dx}\). The equation is then transformed into a variable separable form in terms of \(v\) and \(x\).

Example: Solve \((x^2-y^2)dx = xydy\)

1. Rearrange and Check for Homogeneity: \(\frac{dy}{dx} = \frac{x^2-y^2}{xy}\). The numerator \(x^2-y^2\) and denominator \(xy\) are both homogeneous functions of degree 2.

2. Substitute: Let \(y=vx\), so \(\frac{dy}{dx} = v+x\frac{dv}{dx}\).

3. The equation becomes: \(v+x\frac{dv}{dx} = \frac{x^2-(vx)^2}{x(vx)} = \frac{x^2(1-v^2)}{vx^2} = \frac{1-v^2}{v}\).

4. Separate variables: \(x\frac{dv}{dx} = \frac{1-v^2}{v} - v = \frac{1-v^2-v^2}{v} = \frac{1-2v^2}{v}\).

5. Rearranging gives \(\frac{v}{1-2v^2}dv = \frac{1}{x}dx\).

6. Integrate: \(\int \frac{v}{1-2v^2}dv = \int \frac{1}{x}dx\). To solve the left integral, we can write it as \(-\frac{1}{4}\int \frac{-4v}{1-2v^2}dv\).

7. This yields \(-\frac{1}{4}\ln|1-2v^2| = \ln|x| + C\).

8. Substitute back: Since \(v=y/x\), the final solution is \((1-2\frac{y^2}{x^2})x^4=A\) for some constant A.

4. Non-Homogeneous Equations (Reducible to Homogeneous)

These are of the form \(\frac{dy}{dx} = \frac{ax+by+c}{a'x+b'y+c'}\).

  • Case I: \(\frac{a}{a'} \neq \frac{b}{b'}\)

    Use the substitution \(x = X+h\) and \(y=Y+k\). Choose constants \(h\) and \(k\) such that the constant terms in the numerator and denominator become zero. This transforms the equation into a homogeneous DE in variables \(X\) and \(Y\).

  • Case II: \(\frac{a}{a'} = \frac{b}{b'}\)

    Use the substitution \(t=ax+by\). This reduces the equation to a variable separable form.

Example (Case I): Solve \(\frac{dy}{dx} = \frac{x+y-2}{-x+y-4}\)

1. Check the condition: Here \(a=1, b=1, a'=-1, b'=1\). Since \(\frac{1}{-1} \neq \frac{1}{1}\), this is Case I.

2. Substitution: Let \(x=X+h\) and \(y=Y+k\). Then \(\frac{dY}{dX} = \frac{dy}{dx}\).

3. The DE becomes \(\frac{dY}{dX} = \frac{(X+h)+(Y+k)-2}{-(X+h)+(Y+k)-4} = \frac{X+Y+(h+k-2)}{-X+Y+(-h+k-4)}\).

4. Find h and k: Set the constant parts to zero:
\(h+k-2=0\)
\(-h+k-4=0\)
Solving this system gives \(2k-6=0 \implies k=3\), and \(h+3-2=0 \implies h=-1\).

5. New Homogeneous DE: With \(h=-1, k=3\), the equation is \(\frac{dY}{dX} = \frac{X+Y}{-X+Y}\).

6. Solve the homogeneous DE: Let \(Y=vX\), so \(\frac{dY}{dX}=v+X\frac{dv}{dX}\).
\(v+X\frac{dv}{dX} = \frac{X+vX}{-X+vX} = \frac{1+v}{v-1}\).
\(X\frac{dv}{dX} = \frac{1+v}{v-1} - v = \frac{1+v-v^2+v}{v-1} = \frac{-v^2+2v+1}{v-1}\).

7. Separate and Integrate: \(\int\frac{v-1}{-v^2+2v+1}dv = \int\frac{dX}{X}\). This results in \(-\frac{1}{2}\ln|-v^2+2v+1| = \ln|X| + C\).

8. Back-substitute: Replace \(v\) with \(Y/X\): \((1+2\frac{Y}{X}-\frac{Y^2}{X^2})X^2 = A\).

9. Final Substitution: Replace \(X=x-h=x+1\) and \(Y=y-k=y-3\) to get the final solution in terms of \(x\) and \(y\): \((1 + \frac{2(y-3)}{x+1} - \frac{(y-3)^2}{(x+1)^2})(x+1)^2 = A\).

Lecture 3: First-Order Differential Equations

Reducible to Homogeneous Equations

For equations of the form:

$$\frac{dy}{dx} = \frac{ax+by+c}{a'x+b'y+c'}$$
Case 1: $$\frac{a}{a'} \neq \frac{b}{b'}$$

Substitute $$x = X+h$$ and $$y = Y+k$$. The equation becomes homogeneous:

$$\frac{dY}{dX} = \frac{aX+bY}{a'X+b'Y}$$

Then substitute $$Y=vX$$.

Case 2: $$\frac{a}{a'} = \frac{b}{b'}$$

Use substitution, for example, $$t = ax+by$$.

Example Problem: $$\frac{dy}{dx} = \frac{2x+3y+4}{4x+6y+5}$$

Given: $$(2y+2x+4)dx - (4x+6y+5)dy = 0$$

$$\frac{dy}{dx} = \frac{2x+3y+4}{4x+6y+5}$$

Here, $$a=2, a'=4, b=3, b'=6$$.

$$\frac{a}{a'} = \frac{2}{4} = \frac{1}{2} \quad \text{and} \quad \frac{b}{b'} = \frac{3}{6} = \frac{1}{2}$$

Since $$\frac{a}{a'} = \frac{b}{b'}$$, this is Case 2.

Solution:

Let $$t = 2x+3y$$. Differentiating:

$$\frac{dt}{dx} = 2 + 3\frac{dy}{dx} \implies \frac{dy}{dx} = \frac{1}{3}\left(\frac{dt}{dx} - 2\right)$$

Substituting into the original equation:

$$\frac{1}{3}\left(\frac{dt}{dx} - 2\right) = \frac{t+4}{2t+5}$$

After simplification:

$$\frac{dt}{dx} = \frac{7t+22}{2t+5}$$

Separating variables and integrating leads to the solution.

First Order Linear Differential Equations

Leibnitz Equation

The standard form is:

$$\frac{dy}{dx} + P(x)y = Q(x)$$

where $$P(x)$$ and $$Q(x)$$ are functions of $$x$$.

Solution Method

We use an Integrating Factor (I.F.):

$$\text{I.F.} = e^{\int P(x)dx}$$

The general solution is:

$$y \cdot (\text{I.F.}) = \int Q(x) \cdot (\text{I.F.}) dx + C$$
Example: $$x\frac{dy}{dx} - (y-3x^2) = 0$$

Rearranging: $$x\frac{dy}{dx} - y = -3x^2$$

Dividing by $$x$$:

$$\frac{dy}{dx} - \frac{1}{x}y = -3x$$

This is a first-order linear DE with $$P(x) = -\frac{1}{x}$$ and $$Q(x) = -3x$$.

Step 1: Find the Integrating Factor

$$\text{I.F.} = e^{\int -\frac{1}{x}dx} = e^{-\ln(x)} = \frac{1}{x}$$

Step 2: Apply the solution formula

$$y \cdot \frac{1}{x} = \int (-3x) \cdot \frac{1}{x} dx + C = \int -3 dx + C = -3x + C$$

General Solution: $$\frac{y}{x} + 3x = C$$

Bernoulli's Equation

Non-Linear Equations Reducible to Linear Form

The standard form is:

$$\frac{dy}{dx} + P(x)y = Q(x)y^n$$

Solution Steps:

  1. Divide by $$y^n$$: $$y^{-n}\frac{dy}{dx} + P(x)y^{1-n} = Q(x)$$
  2. Substitute $$z = y^{1-n}$$
  3. Then $$\frac{dz}{dx} = (1-n)y^{-n}\frac{dy}{dx}$$
  4. The equation becomes linear in $$z$$:
    $$\frac{dz}{dx} + (1-n)P(x)z = (1-n)Q(x)$$

Lecture 4: Higher-Order Linear Differential Equations

Higher Order Linear DE

The general form of an $$n^{th}$$ order linear DE is:

$$\frac{d^ny}{dx^n} + P_1(x)\frac{d^{n-1}y}{dx^{n-1}} + P_2(x)\frac{d^{n-2}y}{dx^{n-2}} + \dots + P_n(x)y = Q(x)$$

Linear DE with Constant Coefficients

If $$P_1(x), P_2(x), \dots, P_n(x)$$ are constants ($$k_1, k_2, \dots, k_n$$):

$$\frac{d^ny}{dx^n} + k_1\frac{d^{n-1}y}{dx^{n-1}} + k_2\frac{d^{n-2}y}{dx^{n-2}} + \dots + k_ny = Q(x)$$
Classification
  • If $$Q(x) = 0$$, the equation is Homogeneous
  • If $$Q(x) \neq 0$$, the equation is Non-homogeneous

Solution Structure

The general solution is $$y = y_c + y_p$$ where:

  • $$y_c$$ is the Complementary Function (C.F.) - solution to the homogeneous equation
  • $$y_p$$ is the Particular Integral (P.I.) - particular solution to the non-homogeneous equation

The D-Operator Method

Let $$D = \frac{d}{dx}$$, $$D^2 = \frac{d^2}{dx^2}$$, etc. The DE can be written as:

$$(D^n + k_1D^{n-1} + \dots + k_n)y = Q(x)$$

Or simply $$f(D)y = Q(x)$$.

Finding the Complementary Function ($$y_c$$)

To find $$y_c$$, we solve the homogeneous equation $$f(D)y=0$$.

We form the Auxiliary Equation (A.E.) by replacing D with m:

$$m^n + k_1m^{n-1} + \dots + k_n = 0$$

Cases for Roots of the Auxiliary Equation

Case 1: Real and Distinct Roots

If the roots $$m_1, m_2, \dots, m_n$$ are all real and different:

$$y_c = C_1e^{m_1x} + C_2e^{m_2x} + \dots + C_ne^{m_nx}$$
Example: $$(D^2+5D+6)y=0$$

A.E.: $$m^2 + 5m + 6 = 0$$

Factoring: $$(m+3)(m+2) = 0$$

Roots: $$m_1 = -3$$, $$m_2 = -2$$ (real and distinct)

$$y_c = C_1e^{-3x} + C_2e^{-2x}$$
Case 2: Real and Repeated Roots

If a root $$m$$ is repeated $$n$$ times:

$$y_c = (C_1 + C_2x + C_3x^2 + \dots + C_nx^{n-1})e^{mx}$$
Example: $$\frac{d^2y}{dx^2} + 4\frac{dy}{dx} + 4y = 0$$

A.E.: $$m^2 + 4m + 4 = 0$$

Factoring: $$(m+2)^2 = 0$$

Roots: $$m = -2, -2$$ (repeated root)

$$y_c = (C_1 + C_2x)e^{-2x}$$
Case 3: Complex Conjugate Roots

If the roots are a complex conjugate pair $$m = \alpha \pm i\beta$$:

$$y_c = e^{\alpha x}(C_1\cos(\beta x) + C_2\sin(\beta x))$$
Example: $$\frac{d^2y}{dx^2} + y = 0$$

A.E.: $$m^2 + 1 = 0$$

Roots: $$m = \pm i$$ (complex conjugate pair with $$\alpha=0, \beta=1$$)

$$y_c = C_1\cos(x) + C_2\sin(x)$$

Practice Problems

1. Solve $$\frac{d^2y}{dx^2} + 6\frac{dy}{dx} + 9y = 0$$

A.E.: $$m^2 + 6m + 9 = 0 \implies (m+3)^2 = 0$$

Roots: $$m = -3, -3$$ (Real and repeated)

Solution: $$y = (C_1 + C_2x)e^{-3x}$$

2. Solve $$(D^3 + D^2 + 4D + 4)y = 0$$

A.E.: $$m^3 + m^2 + 4m + 4 = 0$$

Factoring: $$(m^2+4)(m+1) = 0$$

Roots: $$m = -1$$ (real), $$m = \pm 2i$$ (complex, $$\alpha=0, \beta=2$$)

Solution: $$y = C_1e^{-x} + C_2\cos(2x) + C_3\sin(2x)$$

3. Solve $$(D^2+4)y = 0$$

A.E.: $$m^2+4 = 0$$

Roots: $$m = \pm 2i$$ (complex, $$\alpha=0, \beta=2$$)

Solution: $$y = C_1\cos(2x) + C_2\sin(2x)$$

Lecture 5: Complementary Functions & Intro to Particular Integrals

Recap: Finding the Complementary Function ($$y_c$$)

The general solution to a non-homogeneous differential equation is the sum of the complementary function and the particular integral: $$y_{general} = y_{cf} + y_{pi}$$. The complementary function, $$y_{cf}$$, is the solution to the corresponding homogeneous equation, $$P(D)y = 0$$.

Cases for Roots of the Auxiliary Equation

Real and Repeated Roots

If a real root $$m$$ is repeated $$n$$ times, the complementary function is:

$$y_c = (C_1 + C_2x + C_3x^2 + \dots + C_nx^{n-1})e^{mx}$$
Complex Conjugate Roots

If the roots are a complex conjugate pair $$m = \alpha \pm i\beta$$:

$$y_c = e^{\alpha x}(C_1\cos(\beta x) + C_2\sin(\beta x))$$

Examples: Complementary Function

1. Solve $$(D^2 + 6D + 9)y = 0$$

Auxiliary Equation: $$m^2 + 6m + 9 = 0 \implies (m+3)^2 = 0$$

Roots: $$m = -3, -3$$ (Real and repeated)

Solution: $$y_{cf} = (C_1 + C_2x)e^{-3x}$$

2. Solve $$(D^3 + D^2 + 4D + 4)y = 0$$

A.E.: $$m^3 + m^2 + 4m + 4 = 0$$

Factoring: $$m^2(m+1) + 4(m+1) = 0 \implies (m^2+4)(m+1) = 0$$

Roots: $$m = -1$$ and $$m = \pm 2i$$ (complex, with $$\alpha=0, \beta=2$$)

Solution: $$y_{cf} = C_1e^{-x} + C_2\cos(2x) + C_3\sin(2x)$$

3. Solve $$(D^4+4)y = 0$$

A.E.: $$m^4+4=0$$. We complete the square.

$$m^4+4m^2+4 - 4m^2 = 0 \implies (m^2+2)^2 - (2m)^2 = 0$$

Factoring: $$(m^2+2m+2)(m^2-2m+2) = 0$$

Roots: From $$m^2+2m+2=0$$, $$m=-1 \pm i$$. From $$m^2-2m+2=0$$, $$m=1 \pm i$$.

Solution: $$y_{cf} = e^{-x}(C_1\cos(x) + C_2\sin(x)) + e^x(C_3\cos(x) + C_4\sin(x))$$

Finding the Particular Integral ($$y_{pi}$$)

The particular integral is a solution to the non-homogeneous equation $$P(D)y = Q(x)$$. It is represented as:

$$y_{pi} = \frac{1}{P(D)}Q(x)$$

Case 1: When $$Q(x) = e^{ax}$$

Rule: If $$P(a) \neq 0$$, replace D with a.

$$y_{pi} = \frac{1}{P(D)}e^{ax} = \frac{e^{ax}}{P(a)}$$

Case of Failure: If $$P(a) = 0$$, but $$P'(a) \neq 0$$, the rule is:

$$y_{pi} = \frac{xe^{ax}}{P'(a)}$$
Example: Solve $$(D^2+5D+6)y = e^x$$

Complementary Function:

A.E.: $$m^2+5m+6=0 \implies (m+2)(m+3)=0 \implies m=-2, -3$$.

$$y_{cf} = C_1e^{-2x} + C_2e^{-3x}$$

Particular Integral:

Here, $$Q(x) = e^x$$, so $$a=1$$. The operator is $$P(D) = D^2+5D+6$$.

Check the condition: $$P(1) = 1^2 + 5(1) + 6 = 12 \neq 0$$.

$$y_{pi} = \frac{e^x}{P(1)} = \frac{e^x}{12}$$

General Solution:

$$y = y_{cf} + y_{pi} = C_1e^{-2x} + C_2e^{-3x} + \frac{e^x}{12}$$

Lecture 6: Methods for Finding the Particular Integral

Case I: $$Q(x) = e^{ax}$$

This section covers the rules for finding the particular integral when the non-homogeneous term is an exponential function.

Summary of Rules

  • If $$P(a) \neq 0$$, the particular integral is $$y_{pi} = \frac{e^{ax}}{P(a)}$$.
  • If $$P(a) = 0$$ but $$P'(a) \neq 0$$ (single root failure), then $$y_{pi} = \frac{xe^{ax}}{P'(a)}$$.
  • If $$P(a) = 0$$ and $$P'(a) = 0$$ but $$P''(a) \neq 0$$ (double root failure), then $$y_{pi} = \frac{x^2e^{ax}}{P''(a)}$$.
Example: Solve $$(D+2)(D-1)^2 y = e^{-2x} + 2\sinh(x)$$

First, rewrite the equation using the definition $$2\sinh(x) = e^x - e^{-x}$$.

The equation becomes $$(D+2)(D-1)^2 y = e^{-2x} + e^x - e^{-x}$$.

Complementary Function: The roots of the auxiliary equation are $$m=-2, 1, 1$$. The CF is $$y_{cf} = C_1e^{-2x} + (C_2+C_3x)e^x$$.

Particular Integral: We find the PI for each term separately.

  • For $$e^{-2x}$$ ($$a=-2$$): This is a single root failure since $$P(-2)=0$$. We use $$P'(D)=(D-1)^2 + 2(D+2)(D-1)$$, which gives $$P'(-2)=9$$. The PI part is $$\frac{xe^{-2x}}{9}$$.
  • For $$e^x$$ ($$a=1$$): This is a double root failure since $$P(1)=0$$ and $$P'(1)=0$$. We use $$P''(D)=4(D-1)+2(D+2)$$, which gives $$P''(1)=6$$. The PI part is $$\frac{x^2e^x}{6}$$.
  • For $$-e^{-x}$$ ($$a=-1$$): There is no failure since $$P(-1)=(-1+2)(-1-1)^2 = 4 \neq 0$$. The PI part is $$-\frac{e^{-x}}{4}$$.

The total particular integral is $$y_{pi} = \frac{xe^{-2x}}{9} + \frac{x^2e^x}{6} - \frac{e^{-x}}{4}$$.

Case II: $$Q(x) = \sin(ax+b)$$ or $$\cos(ax+b)$$

For sinusoidal functions, the standard method is to replace every instance of $$D^2$$ with $$-a^2$$. This method is valid as long as the denominator does not become zero.

Example: Solve $$(D^3+1)y = \cos(2x+1)$$

1. Set up the PI: $$y_{pi} = \frac{1}{D^3+1}\cos(2x+1)$$.

2. Apply the rule: Here $$a=2$$, so we replace $$D^2$$ with $$-a^2 = -4$$. We write $$D^3$$ as $$D \cdot D^2$$.

$$y_{pi} = \frac{1}{D(D^2)+1}\cos(2x+1) = \frac{1}{D(-4)+1}\cos(2x+1) = \frac{1}{1-4D}\cos(2x+1)$$

3. Rationalize the Operator: To eliminate the $$D$$ in the denominator, multiply the numerator and denominator by the conjugate, $$1+4D$$.

$$y_{pi} = \frac{1+4D}{(1-4D)(1+4D)}\cos(2x+1) = \frac{1+4D}{1-16D^2}\cos(2x+1)$$

4. Apply the rule again: Replace $$D^2$$ with $$-4$$.

$$y_{pi} = \frac{1+4D}{1-16(-4)}\cos(2x+1) = \frac{1+4D}{65}\cos(2x+1)$$

5. Differentiate: Now, apply the operator $$1+4D$$ to the function.

$$y_{pi} = \frac{1}{65}[\cos(2x+1) + 4D(\cos(2x+1))] = \frac{1}{65}[\cos(2x+1) - 8\sin(2x+1)]$$

Case III: $$Q(x) = x^m$$

When the non-homogeneous term is a polynomial of degree $$m$$, the method is to treat the operator fraction as an inverse operator and expand it as a power series in $$D$$ using the binomial theorem.

Example: Solve $$(D^2+D)y = x^2+2x+4$$

1. Set up the PI: $$y_{pi} = \frac{1}{D^2+D}(x^2+2x+4) = \frac{1}{D(1+D)}(x^2+2x+4)$$.

2. Expand the Operator: Use the binomial expansion for $$(1+D)^{-1} = 1 - D + D^2 - \dots$$.

$$y_{pi} = \frac{1}{D}[1-D+D^2-\dots](x^2+2x+4)$$

3. Apply the Operators: Distribute the operators onto the polynomial. We only need terms up to $$D^2$$, as higher derivatives of $$x^2$$ are zero.

$$y_{pi} = \frac{1}{D}[(x^2+2x+4) - (2x+2) + (2)] = \frac{1}{D}[x^2+4]$$

4. Integrate: The operator $$\frac{1}{D}$$ represents integration.

$$y_{pi} = \int (x^2+4)dx = \frac{x^3}{3} + 4x$$

Case IV: $$Q(x) = e^{ax}V(x)$$

This case uses the exponential shift rule. The rule states that you can move the exponential term $$e^{ax}$$ to the left of the operator, provided you replace every $$D$$ in the operator with $$D+a$$.

$$\frac{1}{P(D)}[e^{ax}V(x)] = e^{ax} \frac{1}{P(D+a)}V(x)$$

Case V: $$Q(x) = xV(x)$$

When the function is a product of $$x$$ and another function $$V(x)$$, a special formula derived from the Leibniz theorem is used.

$$\frac{1}{P(D)}[xV(x)] = \left[x - \frac{P'(D)}{P(D)}\right]\frac{1}{P(D)}V(x)$$
Example: Solve $$(D+1)^2y = x\cos(x)$$

Here, $$V(x)=\cos(x)$$, $$P(D)=(D+1)^2$$, and $$P'(D)=2(D+1)$$.

1. Apply the Formula:

$$y_{pi} = \left[x - \frac{2(D+1)}{(D+1)^2}\right]\frac{1}{(D+1)^2}\cos(x) = \left[x - \frac{2}{D+1}\right]\frac{1}{D^2+2D+1}\cos(x)$$

2. Evaluate the Inner Operator: First, find $$\frac{1}{D^2+2D+1}\cos(x)$$. Using the rule for cosines ($$D^2 \to -1^2=-1$$):

$$\frac{1}{-1+2D+1}\cos(x) = \frac{1}{2D}\cos(x) = \frac{1}{2}\int \cos(x)dx = \frac{1}{2}\sin(x)$$

3. Substitute Back:

$$y_{pi} = \left[x - \frac{2}{D+1}\right]\left(\frac{1}{2}\sin(x)\right) = \frac{x}{2}\sin(x) - \frac{1}{D+1}\sin(x)$$

4. Evaluate the Remaining Part: Solve $$\frac{1}{D+1}\sin(x)$$ by rationalizing:

$$\frac{D-1}{(D+1)(D-1)}\sin(x) = \frac{D-1}{D^2-1}\sin(x) = \frac{D-1}{-1-1}\sin(x) = -\frac{1}{2}(\cos(x) - \sin(x))$$

5. Final Answer: Combine the parts.

$$y_{pi} = \frac{x}{2}\sin(x) - \left(-\frac{1}{2}(\cos(x) - \sin(x))\right) = \frac{x}{2}\sin(x) + \frac{1}{2}\cos(x) - \frac{1}{2}\sin(x)$$

Lecture 7: Advanced Methods for Particular Integrals

Recap: Methods for Finding Particular Integral ($$y_{pi}$$)

  • Case 1: $$Q(x) = e^{ax}$$: $$y_{pi} = \frac{e^{ax}}{P(a)}$$, with failure cases handled by multiplying by $$x$$ and using derivatives of $$P(D)$$.
  • Case 2: $$Q(x) = \sin(ax+b)$$ or $$\cos(ax+b)$$: Replace $$D^2$$ with $$-a^2$$.
  • Case 3: $$Q(x) = x^m$$: Expand $$[P(D)]^{-1}$$ using a binomial series.
  • Case 4: $$Q(x) = e^{ax}V(x)$$: Use the exponential shift rule $$y_{pi} = e^{ax}\frac{1}{P(D+a)}V(x)$$.
  • Case 5: $$Q(x) = xV(x)$$: Use the formula $$y_{pi} = \left[x - \frac{P'(D)}{P(D)}\right]\frac{1}{P(D)}V(x)$$.

Method of Variation of Parameters

This is a powerful method for finding the particular integral, especially for functions like $$\sec(x), \tan(x), \log(x)$$, or when the DE has variable coefficients. It applies to second-order linear DEs of the form:

$$\frac{d^2y}{dx^2} + P(x)\frac{dy}{dx} + Q(x)y = \phi(x)$$

If the complementary function is $$y_{cf} = C_1y_1(x) + C_2y_2(x)$$, the particular integral is given by:

$$y_{pi} = -y_1 \int \frac{y_2 \phi(x)}{W} dx + y_2 \int \frac{y_1 \phi(x)}{W} dx$$

Where W is the Wronskian of $$y_1$$ and $$y_2$$, defined as the determinant:

$$W = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} = y_1y_2' - y_2y_1'$$
Example: Solve $$\frac{d^2y}{dx^2} + 4y = \tan(2x)$$

1. Find Complementary Function ($$y_{cf}$$):

The auxiliary equation is $$m^2+4=0$$, which gives roots $$m = \pm 2i$$.

$$y_{cf} = C_1\cos(2x) + C_2\sin(2x)$$. Thus, we have $$y_1 = \cos(2x)$$ and $$y_2 = \sin(2x)$$.

2. Calculate the Wronskian (W):

$$y_1' = -2\sin(2x)$$ and $$y_2' = 2\cos(2x)$$.

$$W = (\cos(2x))(2\cos(2x)) - (\sin(2x))(-2\sin(2x)) = 2\cos^2(2x) + 2\sin^2(2x) = 2$$.

3. Find Particular Integral ($$y_{pi}$$):

Here, $$\phi(x) = \tan(2x)$$. Using the formula:

$$y_{pi} = -\cos(2x) \int \frac{\sin(2x)\tan(2x)}{2} dx + \sin(2x) \int \frac{\cos(2x)\tan(2x)}{2} dx$$

Evaluating the integrals leads to:

$$y_{pi} = -\frac{\cos(2x)}{2}\left[\frac{1}{2}\log|\sec(2x)+\tan(2x)| - \frac{\sin(2x)}{2}\right] - \frac{1}{4}\sin(2x)\cos(2x)$$

Method of Undetermined Coefficients

This method applies to linear DEs with constant coefficients. It involves assuming a trial solution for the particular integral based on the form of $$Q(x)$$.

Trial Solutions

If $$Q(x)$$ is... The Trial Solution $$y_{pi}$$ is...
$$k e^{ax}$$ $$C e^{ax}$$
$$k x^m$$ (a polynomial of degree m) $$C_m x^m + C_{m-1}x^{m-1} + \dots + C_1 x + C_0$$
$$k\cos(ax+b)$$ or $$k\sin(ax+b)$$ $$C_1\cos(ax+b) + C_2\sin(ax+b)$$

🚨 Modification Rule

If any term in the assumed trial solution is already part of the complementary function ($$y_{cf}$$), you must multiply the entire trial solution by $$x$$. If this new solution still duplicates a term (e.g., for a double root), multiply by $$x$$ again (i.e., by $$x^2$$).

Example: Solve $$\frac{d^2y}{dx^2} + 2\frac{dy}{dx} + 4y = 2x^2 + 4e^{-x}$$

1. Find CF: The A.E. $$m^2+2m+4=0$$ has roots $$m = -1 \pm i\sqrt{3}$$.

$$y_{cf} = e^{-x}[C_1\cos(\sqrt{3}x) + C_2\sin(\sqrt{3}x)]$$.

2. Assume Trial PI: For $$Q(x) = 2x^2 + 4e^{-x}$$, the trial solution is the sum of the individual forms.

$$y_{pi} = (C_0 + C_1x + C_2x^2) + C_3e^{-x}$$. No term duplicates the CF.

3. Find Coefficients: Substitute $$y_{pi}$$, $$y_{pi}'$$, and $$y_{pi}''$$ into the DE and compare coefficients of like terms ($$x^2, x, 1, e^{-x}$$).

This yields $$C_2 = 1/2, C_1 = -1/2, C_0 = 0, C_3 = 4/3$$.

$$y_{pi} = -\frac{1}{2}x + \frac{1}{2}x^2 + \frac{4}{3}e^{-x}$$.

4. General Solution: $$y = y_{cf} + y_{pi}$$.

Linear DEs with Variable Coefficients

Certain DEs with variable coefficients, such as the Cauchy-Euler and Legendre equations, can be reduced to equations with constant coefficients.

Cauchy's Homogeneous (Euler) Equation

This equation has the form:

$$x^n \frac{d^ny}{dx^n} + k_1 x^{n-1} \frac{d^{n-1}y}{dx^{n-1}} + \dots + k_n y = \phi(x)$$

We solve it using the substitution $$x = e^t$$, which implies $$t = \log(x)$$. This transforms the differential operator:

$$x\frac{dy}{dx} = \frac{dy}{dt} = Dy, \text{ where } D = \frac{d}{dt}$$

Lecture 8: Variable Coefficient DEs and Intro to PDEs

Cauchy-Euler Equation

This equation is transformed into a constant coefficient DE by using the substitution $$x=e^t$$. The operator transformations are:

  • $$x\frac{dy}{dx} = Dy$$
  • $$x^2\frac{d^2y}{dx^2} = D(D-1)y$$
  • $$x^3\frac{d^3y}{dx^3} = D(D-1)(D-2)y$$, and so on.
Example: Solve $$x^2\frac{d^2y}{dx^2} - x\frac{dy}{dx} + y = \log(x)$$

1. Substitute and Transform:

Using $$x=e^t$$, $$t=\log(x)$$, and the operator transformations, the equation becomes:

$$[D(D-1)y] - [Dy] + y = t$$

Simplifying gives $$(D^2 - 2D + 1)y = t$$, or $$(D-1)^2y = t$$.

2. Solve the Constant Coefficient Equation (in t):

The A.E. is $$(m-1)^2=0 \implies m=1,1$$. The complementary function is $$y_{cf} = (C_1 + C_2t)e^t$$.

For the PI, $$y_{pi} = \frac{1}{(D-1)^2}t = (1-D)^{-2}t = (1+2D+\dots)t = t+2$$.

The solution in terms of t is $$y = (C_1 + C_2t)e^t + t + 2$$.

3. Substitute Back to x:

Replace $$t$$ with $$\log(x)$$ and $$e^t$$ with $$x$$:

$$y(x) = (C_1 + C_2\log(x))x + \log(x) + 2$$.

Legendre's Linear Equation

This is a generalization of the Cauchy-Euler equation, with the form:

$$(ax+b)^n \frac{d^ny}{dx^n} + k_1 (ax+b)^{n-1} \frac{d^{n-1}y}{dx^{n-1}} + \dots + k_n y = Q(x)$$

It is solved using the substitution $$ax+b = e^t$$. The operators transform as follows:

  • $$(ax+b)\frac{dy}{dx} = aDy$$
  • $$(ax+b)^2\frac{d^2y}{dx^2} = a^2D(D-1)y$$
  • $$(ax+b)^n\frac{d^ny}{dx^n} = a^nD(D-1)\dots(D-n+1)y$$
Example: Solve $$(1+x)^2\frac{d^2y}{dx^2} + (1+x)\frac{dy}{dx} + y = 2\sin(\log(1+x))$$

1. Transform: This is a Legendre equation with $$a=1, b=1$$. Let $$1+x = e^t$$. The DE becomes:

$$[D(D-1)y] + [Dy] + y = 2\sin(t)$$

This simplifies to $$(D^2+1)y = 2\sin(t)$$.

2. Solve in t: The A.E. $$m^2+1=0$$ gives $$m=\pm i$$. So, $$y_{cf} = C_1\cos(t) + C_2\sin(t)$$.

The PI involves a case of failure since the operator $$D^2+1$$ annihilates $$\sin(t)$$. The PI is $$y_{pi} = -t\cos(t)$$.

3. Substitute back to x: The final solution is:

$$y(x) = C_1\cos(\log(1+x)) + C_2\sin(\log(1+x)) - \log(1+x)\cos(\log(1+x))$$

Introduction to Partial Differential Equations (PDEs)

A Partial Differential Equation (PDE) is an equation that involves partial derivatives of one or more dependent variables with respect to two or more independent variables.

Basic Definitions

  • Order: The order of the highest partial derivative in the equation.
  • Degree: The power of the highest-order derivative after the equation has been cleared of any radicals or fractions with respect to the derivatives.

Classification of PDEs by Linearity

Linear PDE
The dependent variable and all its derivatives appear only to the first degree and are not multiplied together. Example: $$\frac{\partial u}{\partial x} + \frac{\partial u}{\partial y} = 2$$
Semilinear PDE
The highest-order derivatives appear linearly, but lower-order terms can be non-linear. Example: $$\frac{\partial^2 u}{\partial x^2} + (\frac{\partial u}{\partial y})^2 + u = 2$$
Quasi-linear PDE
The highest-order derivatives are linear, but their coefficients can be functions of the dependent variable or lower-order derivatives. Example: $$u\frac{\partial u}{\partial y} + u = 0$$
Non-linear PDE
A PDE that is not linear in the highest-order derivatives. Example: $$(\frac{\partial y}{\partial x})^2 + \frac{\partial y}{\partial t} = 2$$

Lecture 9: Solving Partial Differential Equations

Solving PDEs by Direct Integration

For some simple PDEs, a solution can be found by integrating the equation directly. When integrating with respect to one variable, the "constant" of integration is an arbitrary function of the other variable(s).

Example: Solve $$\frac{\partial^2 z}{\partial x^2} = xy^2$$

1. First Integration:

Integrate with respect to $$x$$, treating $$y$$ as a constant.

$$\frac{\partial z}{\partial x} = \int xy^2 dx = \frac{x^2}{2}y^2 + f(y)$$

Here, $$f(y)$$ is an arbitrary function of $$y$$ that acts as the constant of integration.

2. Second Integration:

Integrate again with respect to $$x$$.

$$z = \int \left(\frac{x^2}{2}y^2 + f(y)\right) dx = \frac{x^3}{6}y^2 + x f(y) + g(y)$$

Here, $$g(y)$$ is another arbitrary function of $$y$$, representing the new constant of integration.

First-Order Quasi-linear PDEs: Lagrange's Method

A first-order quasi-linear PDE has the general form:

$$P(x,y,z)\frac{\partial z}{\partial x} + Q(x,y,z)\frac{\partial z}{\partial y} = R(x,y,z)$$

This type of equation is solved using Lagrange's method of auxiliary equations. The auxiliary equations are:

$$\frac{dx}{P} = \frac{dy}{Q} = \frac{dz}{R}$$

By solving this system, we find two independent solutions, $$u_1(x,y,z) = c_1$$ and $$u_2(x,y,z) = c_2$$. The general solution is then an arbitrary function relating these two solutions, often written as $$F(u_1, u_2) = 0$$.

Example: Solve $$x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = 3z$$

1. Set up Auxiliary Equations:

Here, $$P=x, Q=y, R=3z$$. The equations are:

$$\frac{dx}{x} = \frac{dy}{y} = \frac{dz}{3z}$$

2. Find the First Solution ($$u_1$$):

Take the first two fractions and integrate:

$$\int\frac{dx}{x} = \int\frac{dy}{y} \implies \ln(x) = \ln(y) + \ln(c_1) \implies u_1 = \frac{x}{y} = c_1$$

3. Find the Second Solution ($$u_2$$):

Take the second and third fractions and integrate:

$$\int\frac{dy}{y} = \int\frac{dz}{3z} \implies \ln(y) = \frac{1}{3}\ln(z) + \ln(c_2) \implies 3\ln(y) = \ln(z) + \ln(c_2^3) \implies u_2 = \frac{y^3}{z} = c_2'$$

4. Write the General Solution:

The general solution is an arbitrary function of $$u_1$$ and $$u_2$$.

$$F\left(\frac{x}{y}, \frac{y^3}{z}\right) = 0$$

Lagrange's Method of Multipliers

When the variables in the auxiliary equations cannot be easily separated, we can use the method of multipliers. For any multipliers $$l, m, n$$, the following relation holds:

$$\frac{dx}{P} = \frac{dy}{Q} = \frac{dz}{R} = \frac{l\;dx + m\;dy + n\;dz}{lP + mQ + nR}$$

The strategy is to choose multipliers $$l, m, n$$ such that the denominator becomes zero: $$lP + mQ + nR = 0$$. If the denominator is zero, the numerator must also be zero, which gives us a new integrable equation: $$l\;dx + m\;dy + n\;dz = 0$$.

Second-Order Linear PDEs

Classification

A general second-order linear PDE in two variables has the form:

$$A\frac{\partial^2 u}{\partial x^2} + B\frac{\partial^2 u}{\partial x \partial y} + C\frac{\partial^2 u}{\partial y^2} + D\frac{\partial u}{\partial x} + E\frac{\partial u}{\partial y} + Fu = G(x,y)$$

The equation is classified based on the value of the discriminant $$B^2 - 4AC$$:

  • If $$B^2 - 4AC > 0$$, the PDE is Hyperbolic (e.g., wave equation).
  • If $$B^2 - 4AC = 0$$, the PDE is Parabolic (e.g., heat equation).
  • If $$B^2 - 4AC < 0$$, the PDE is Elliptic (e.g., Laplace's equation).

Method of Separation of Variables

This method is used to solve homogeneous linear PDEs. It works by assuming the solution can be expressed as a product of functions, each depending on only one of the independent variables. For a function $$z(x,y)$$, we assume a solution of the form $$z(x,y) = X(x)Y(y)$$.

Example: Solve $$\frac{\partial^2 z}{\partial x^2} + 4\frac{\partial^2 z}{\partial y^2} = 0$$

1. Classify the PDE:

Here, $$A=1, B=0, C=4$$. The discriminant is $$B^2-4AC = 0 - 4(1)(4) = -16 < 0$$, so the equation is elliptic.

2. Assume and Substitute the Solution:

Assume $$z(x,y) = X(x)Y(y)$$. Substituting into the PDE gives:

$$X''(x)Y(y) + 4X(x)Y''(y) = 0$$

3. Separate the Variables:

Rearrange the equation to separate the functions of $$x$$ and $$y$$:

$$\frac{X''}{X} = -4\frac{Y''}{Y}$$

Since the left side depends only on $$x$$ and the right side only on $$y$$, both must be equal to a constant, say $$k$$.

4. Solve the ODEs:

This gives us two ordinary differential equations:

  • $$\frac{X''}{X} = k \implies X'' - kX = 0$$. The solution is $$X(x) = a_1e^{\sqrt{k}x} + a_2e^{-\sqrt{k}x}$$.
  • $$-4\frac{Y''}{Y} = k \implies 4Y'' + kY = 0$$. The solution is $$Y(y) = b_1\cos\left(\frac{\sqrt{k}}{2}y\right) + b_2\sin\left(\frac{\sqrt{k}}{2}y\right)$$.

5. Form the General Solution:

The complete solution is the product $$z(x,y) = X(x)Y(y)$$. More general solutions are formed by taking a superposition (sum) of these solutions for different values of k. The specific values for the constants are found using initial or boundary conditions.

Lecture 10: Formation of Partial Differential Equations

1. By Eliminating Arbitrary Constants

A PDE can be formed by differentiating a relation to eliminate the arbitrary constants present.

Example 1

Form a PDE by eliminating the constants a and b from the equation: $$2z = \frac{x^2}{a^2} + \frac{y^2}{b^2}$$

  • Step 1: Differentiate with respect to x
    Differentiate the equation partially with respect to \(x\), keeping \(y\) constant. $$2\frac{\partial z}{\partial x} = \frac{2x}{a^2}$$ Rearranging gives an expression for \(\frac{1}{a^2}\): $$\frac{1}{a^2} = \frac{1}{x}\frac{\partial z}{\partial x}$$
  • Step 2: Differentiate with respect to y
    Differentiate partially with respect to \(y\), treating \(x\) as a constant. $$2\frac{\partial z}{\partial y} = \frac{2y}{b^2}$$ Rearranging gives an expression for \(\frac{1}{b^2}\): $$\frac{1}{b^2} = \frac{1}{y}\frac{\partial z}{\partial y}$$
  • Step 3: Substitute and Simplify
    Substitute the expressions for \(\frac{1}{a^2}\) and \(\frac{1}{b^2}\) back into the original equation: $$2z = x^2 \left(\frac{1}{x}\frac{\partial z}{\partial x}\right) + y^2 \left(\frac{1}{y}\frac{\partial z}{\partial y}\right)$$ This simplifies to the final PDE:
    $$2z = x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y}$$

2. By Eliminating an Arbitrary Function

A PDE can also be formed by eliminating an arbitrary function from a relation.

Example 2

Form a PDE by eliminating the function f from: $$z = f(x^2 + y^2)$$

  • Step 1: Differentiate with respect to x $$\frac{\partial z}{\partial x} = f'(x^2 + y^2) \cdot (2x)$$
  • Step 2: Differentiate with respect to y $$\frac{\partial z}{\partial y} = f'(x^2 + y^2) \cdot (2y)$$
  • Step 3: Eliminate the function f'
    By dividing the two resulting equations, the arbitrary function \(f'\) is eliminated: $$\frac{\frac{\partial z}{\partial x}}{\frac{\partial z}{\partial y}} = \frac{2x \cdot f'(x^2+y^2)}{2y \cdot f'(x^2+y^2)} = \frac{x}{y}$$ Cross-multiplying gives the required PDE:
    $$y\frac{\partial z}{\partial x} = x\frac{\partial z}{\partial y}$$

The notes indicate that for these examples, the resulting PDE is of the first order.
